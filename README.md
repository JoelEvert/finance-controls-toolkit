# Finance Automation Library

A bank of pre-built finance automations you can take apart, edit, and run. No frameworks, no signups — Python scripts that work on CSV exports from any ERP.

## Structure

```
checks/       Small, single-purpose control checks (run on an AP/GL export)
retrieval/    Playwright scripts that fetch data from portals (coming)
models/       Scripts that generate Excel models with live formulas (coming)
workflows/    Pipelines chaining retrieval -> checks -> models (coming)
tools/        Sample data generators and shared utilities
```

## The unit format

Every automation is one folder:

- `SKILL.md` — what it finds, why it matters, how to run and tune it, known noise sources
- `check.py` (or `script.py`) — the code, with all config constants at the top
- Findings output as CSV next to the script

## Quick start

```
pip install pandas
cd checks
python run_all.py                      # runs all checks on bundled sample data
python run_all.py your_ap_export.csv   # runs them on your own export
```

The sample data (`checks/sample_data/ap_ledger.csv`) is generated by `tools/make_sample_data.py` with every anomaly deliberately planted — so you can see each check catch something before trusting it on real data.

## Design rules

1. Runs on CSV first; API/portal integrations are optional extras.
2. Every config knob is a named constant at the top of the script.
3. Every SKILL.md documents known false-positive sources honestly.
4. pandas + stdlib only for checks. No dependency sprawl.

## Checks included

| Check | Catches |
|---|---|
| duplicate-payments | Same vendor + amount paid twice within days |
| vendor-name-fuzzing | "Nordic Supplies ApS" vs "Nordic Supplies APS" |
| round-numbers | Vendors billing suspiciously round amounts |
| threshold-splitting | Purchases split to stay under approval limits |
| off-hours-postings | Weekend / late-night entries |
| sequence-gaps | Missing invoice numbers = unrecorded liabilities |
| benfords-law | First-digit distribution deviations (fabrication tell) |
